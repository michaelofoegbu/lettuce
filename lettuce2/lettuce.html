FAQ

What can lettuce swap do outside been a Dex?

Lettuce swap can help track and mange your portfolio. You can execute transactions across Arbitrum and bridge platforms that supports Arbitrum Network.

Will the price of my NFT be displayed?

As time goes on adequate display will be made as we intend to expand our tools to showing how much your NFT is worth.

What are the features to expect when using LettuceSwap?

Lettuce has a number of features to make transactions easier including,
Monitor Arbitium Ecosystem yield opportunities
Creating lottery tickets for the community
NFT Gallery (View/manage/share NFTs on Arbitium
Dashboard positions, charts and portfolio visualization

Lettuce Staking?

As much as farming we also welcome staking so as to make users earn more interest rates, we allow users to stake out token to earn immediate rewards all staking features will be accessible as soon as we release mainnet users can start and manage positions without issues
